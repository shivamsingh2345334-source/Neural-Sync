# ⚙️ Neural-Sync — Predictive Maintenance AI

Neural-Sync is an AI-powered predictive maintenance system for industrial motors.

## Features
- Sensor-based anomaly detection (Autoencoder)
- Failure prediction using LSTM
- Remaining Useful Life (RUL) estimation
- Real-time Streamlit dashboard

## Tech Stack
- TensorFlow / Keras
- LSTM Forecasting
- Autoencoders
- Streamlit Deployment

## Dashboard Output
Predicts motor failure probability in next 48 hours with alerts.
